# itd baner

Этот скрипт Python преобразует любое изображение (`input.png`) в пиксели и генерирует JS-код, который рисует эти пиксели на `<canvas>` в браузере.  
Использует RGB цвета исходного изображения, шаг и размер точки можно настраивать для баланса между качеством и скоростью.

---

## Возможности

- Пиксельный рендеринг с точным цветом каждого пикселя  
- Настраиваемый шаг пикселя (`PIXEL_STEP`) для ускорения генерации  
- Настраиваемый размер точки (`DOT_SIZE`) для эффекта «кисти»  
- Автоматическое масштабирование и центрирование изображения на canvas (800×267 px)  
- Генерация самодостаточного JS, который рисует напрямую через `ctx.fillRect`

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/yourusername/pixel-to-canvas-js.git
cd pixel-to-canvas-js
```

2. Установите зависимости Python:

```bash
pip install opencv-python numpy
```

---

## Использование

1. Поместите изображение в папку проекта и назовите его `input.png`.  
2. Откройте `main.py` и при необходимости настройте параметры:

```python
CANVAS_W = 800     # ширина canvas
CANVAS_H = 267     # высота canvas
PIXEL_STEP = 1     # шаг по пикселям (1 = максимально точно, 2-3 = быстрее)
DOT_SIZE = 1       # размер точки на canvas (px)
```

3. Запустите скрипт:

```bash
python main.py
```

4. После выполнения появится файл `output.js`.  
5. Вставьте `output.js` в консоль браузера на сайте с canvas:

```html
<canvas class="drawing-canvas" width="800" height="267"></canvas>
```

JS автоматически нарисует изображение на canvas.

---

## Настройки

| Параметр       | Описание |
|----------------|----------|
| `PIXEL_STEP`   | Шаг по пикселям. Меньше = выше точность, больше = быстрее генерация |
| `DOT_SIZE`     | Размер каждой точки. Можно увеличить для эффекта кисти |
| `CANVAS_W/H`   | Размер canvas. Изображение масштабируется под эти размеры |
